CSS Cheatsheet

info about cheatsheet here

Purple Text

This is how you turn header text purple.

  
    header {
        color: #800080;
    }

Dashed Border

This is how you add a 2px wide dashed border to a container.

  
    div {
        border-style: dashed;
        border-width: 2px
    }

Box Shadow

This is how you add a shadow to a box. X-offset Y-Offset Color

  
    div {
        box-shadow: 10px 5px #78f542;
    }

Media Screen Query

This is how adapt your screen to different browser sizes

  
    @media screen and (max-width: 768px) {
        main,
        nav {
        flex-direction: column;
        }
    }
